## Iniciando projeto

Clone esse projeto e acesse a pasta.

```bash
$ https://github.com/RafaelSoares12/blogDjango.git
$ cd blogDjango
```

Siga os passos:
```bash
# Instando pacotes
$ pip install -r requirements.txt


# Iniciando máquina virtual - Há outras escolha, como o conda.
$ pip install virtualenv
$ python3 -m virtualenv env
$ source env/bin/activate

# Iniciando django
$ python manage.py runserver
```
The app will be available for access on your browser at http://127.0.0.1:8000/
